# Intellij 常用設定

## 快捷鍵

### 選擇文字大小寫
```shell
ctrl + shift + u
```

### 選擇文字駝峰化
需先安裝 ***CamelCase*** plugin
```shell
shift + alt + u
```

###  Code completion vs Cyclic expand word
* Code completion : 會有下拉選單讓你選建議
* Cyclic expand word : 自動幫你補完後面的字(通常都不是....)
