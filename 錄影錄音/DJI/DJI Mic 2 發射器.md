
# 只有 mic 2 發射器 如何充電
如果您手邊只有 DJI Mic 2 發射器 (TX) 而沒有充電盒，您仍然可以輕鬆為其充電。

## 充電方式

- 使用 USB-C 連接埠直接充電：發射器側邊配有一個 USB-C 數據連接埠，除了用於導出錄音檔或升級韌體外，也具備充電功能 [1, 2, 3] 。
- 連接電源：您可以使用隨附的 USB-C 充電線（或任何合格的 Type-C 線材），將其連接至 5V/1A 的電源適配器、電腦 USB 孔或行動電源 [4, 5, 6] 。

## 重要規格與注意事項

- 推薦充電功率：當單獨為發射器充電時，DJI 官方推薦使用 5V/1A 的充電規格 [5, 6] 。
- 充電時間：將發射器單獨充滿電約需 70 分鐘 [5] 。
- 狀態指示燈：充電時，系統狀態指示燈會顯示充電狀態，充滿後指示燈狀態會發生變化（通常為熄滅或變色，視具體固件版本而定） [7, 8] 。

如果您有購買單個發射器的「珍珠白」或「透明黑」版本，包裝內通常也會附帶相關的線材供您直接充電 [9, 10] 。

您目前的發射器是打算透過 藍牙 直接連接手機使用，還是有其他的 接收器 (RX) 需要配對呢？

## 如何接電腦

可以，DJI Mic 2 發射器 (TX) 可以接電腦，但根據您的用途，連接方式與功能會有所不同：

## 1. 讀取錄音檔（有線連接）

如果您要匯出發射器內部的 備份錄音檔，請使用隨附的 USB-C 線直接將發射器與電腦連接 [1, 2] 。

- 功能：電腦會將發射器識別為一個「隨身碟」。
- 路徑：您可以直接在磁碟中找到 WAV 格式的音訊檔 [3, 4] 。
- 其他功能：此模式下也可以進行韌體升級 [5, 6] 。

## 2. 當作電腦麥克風（藍牙連接）

如果您想直接把發射器當作電腦的錄音裝置（例如進行 Zoom 會議或錄音），通常有兩種做法：

- 藍牙直接連線：DJI Mic 2 發射器支援 藍牙直接連接 電腦（需電腦具備藍牙功能） [7, 8] 。
    
    - 步驟：長按發射器錄音鍵 3 秒進入藍牙配對模式（藍色指示燈快閃），接著在電腦的藍牙設定中搜尋並配對即可 [2, 7] 。
    
- 透過接收器 (RX) 連接（最推薦）：如果您有接收器，將接收器透過 USB-C 接上電腦，電腦會自動將其辨識為麥克風 [9, 10] 。這種方式比藍牙連線更穩定，且延遲更低。

## 3. 注意事項

- USB-C 直接錄音限制：在某些系統環境下，單純用 USB-C 線連接發射器，電腦可能只會將其視為磁碟而非麥克風輸入 [2, 4] 。若要進行即時收音，建議優先使用 藍牙 或透過 接收器。
- 32-bit Float：若您是使用接收器連接電腦，還能支援 32-bit Float 錄製，避免音訊過大失真 [11] 。
